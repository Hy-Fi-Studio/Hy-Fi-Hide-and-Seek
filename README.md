# Hy-Fi-Hide-and-Seek
Hy-Fi Hide and Seek is a social minigame where players hide as objects or entities while seekers hunt them down. Includes disguise mechanics, timed rounds, score rewards, and map configs. Ideal for casual servers and creative multiplayer communities.
